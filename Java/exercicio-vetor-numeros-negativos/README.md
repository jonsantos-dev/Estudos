# Exercício: Números Negativos (Vetores)

Este programa lê um número inteiro positivo N (até 10) e armazena N números inteiros em um vetor. Ao final, o sistema filtra e exibe apenas os números negativos informados.

## 📋 Funcionalidades

- Validação de entrada para garantir que o tamanho do vetor esteja entre 1 e 10.
- Armazenamento dinâmico de dados em arrays (vetores).
- Filtro de exibição baseado em condições lógicas (apenas valores menores que 0).

## 🚀 Conceitos Aplicados

- **Vetores (Arrays)**: Criação e manipulação de estruturas de dados indexadas.
- **Laços de Repetição**: Uso de `for` para percorrer o vetor e `do-while` para validação.
- **Estruturas Condicionais**: Verificação de números negativos e intervalo de segurança.
- **Tratamento de Scanner**: Limpeza de buffer com `sc.nextLine()` após leitura de inteiros.

## 🛠️ Tecnologias
