# Exercício: Números Pares (Vetores)

Este programa recebe N números inteiros, armazena-os em um vetor e identifica quais deles são pares. Ao final, o sistema lista esses números e exibe a contagem total de pares encontrados.

## 📋 Funcionalidades

- Leitura de uma quantidade personalizada de números inteiros.
- Identificação de números pares utilizando o operador de módulo.
- Listagem dos valores pares em uma única linha para melhor visualização.
- Exibição do total acumulado de números pares.

## 🚀 Conceitos Aplicados

- **Arrays (Vetores)**: Armazenamento e percorrimento de coleções de dados.
- **Operadores Aritméticos**: Uso do operador `%` (resto da divisão) para lógica de paridade.
- **Contadores**: Implementação de variável incremental para estatísticas do conjunto de dados.
- **Estruturas de Controle**: Uso de laços `for` e condicionais `if` para processamento de listas.
