# Exercício: Abaixo da Média (Vetores)

Este programa lê um número inteiro N e depois um vetor de N números reais. O sistema calcula a média aritmética de todos os elementos e, em seguida, exibe apenas os valores do vetor que estão abaixo dessa média.

## 📋 Funcionalidades

- Leitura de uma quantidade variável de números reais (`double`).
- Cálculo dinâmico da média aritmética do conjunto de dados.
- Filtragem de elementos com base em critério estatístico (abaixo da média).
- Exibição dos resultados com formatação de 3 casas decimais.

## 🚀 Conceitos Aplicados

- **Arrays (Vetores)**: Armazenamento e acesso aos dados para múltiplas iterações.
- **Processamento em Etapas**: Necessidade de concluir o somatório total antes de realizar a filtragem individual.
- **Estruturas de Repetição**: Uso de laços `for` para entrada de dados e posterior verificação condicional.
- **Saída Formatada**: Uso de `printf` para garantir a precisão exigida no exercício.
