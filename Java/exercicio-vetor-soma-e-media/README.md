# Exercício: Soma e Média com Vetores

Este programa recebe uma quantidade N de números inteiros, armazena-os em um vetor e realiza o cálculo do somatório e da média aritmética simples.

## 📋 Funcionalidades

- Leitura de quantidade variável de elementos via terminal.
- Armazenamento de dados em estrutura de Array (Vetor).
- Exibição de todos os elementos inseridos em uma única linha.
- Cálculo de soma total e média com duas casas decimais.

## 🚀 Conceitos Aplicados

- **Arrays (Vetores)**: Uso de `.length` para iterar dinamicamente.
- **Acumuladores**: Variável `soma` incrementada dentro de um laço `for`.
- **Casting**: Conversão de tipos (`int` para `double`) para garantir a precisão no cálculo da média.
- **Interatividade**: Formatação de saída com `printf` e uso de `Locale.US`.

## 🛠️ Tecnologias

- Java 17
- VS Code / Git Bash
