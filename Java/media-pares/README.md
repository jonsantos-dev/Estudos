# Exercício: Média de Pares (Vetores)

Este programa lê um vetor de N números inteiros e calcula a média aritmética apenas dos elementos que forem pares. Caso nenhum número par seja detectado, o sistema informa ao usuário.

## 📋 Funcionalidades

- Leitura de uma quantidade personalizada de números inteiros via console.
- Filtro dinâmico para identificação de números pares.
- Cálculo da média aritmética com validação de existência de dados (evita divisão por zero).
- Exibição do resultado formatado com uma casa decimal.

## 🚀 Conceitos Aplicados

- **Arrays (Vetores)**: Estrutura para armazenamento temporário de múltiplos dados inteiros.
- **Operador de Módulo**: Uso do `% 2 == 0` para lógica de paridade.
- **Tratamento de Fluxo**: Uso de condicionais para validar a presença de números pares antes do cálculo da média.
- **Acumuladores e Contadores**: Gerenciamento de soma e quantidade dentro de um mesmo laço `for`.
