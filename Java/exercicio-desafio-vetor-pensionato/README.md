# Exercício: Pensionato (Vetor de Objetos)

Este programa simula o gerenciamento de aluguel de quartos em um pensionato. O sistema registra o nome, email e o número do quarto escolhido pelo estudante (de 0 a 9), gerando um relatório final dos quartos ocupados em ordem crescente.

## 📋 Funcionalidades

- Cadastro de inquilinos com validação de dados via laço `do-while`.
- Alocação direta no vetor utilizando o número do quarto como índice.
- Verificação de posições nulas para exibição apenas de quartos ocupados.
- Uso de `toString()` para formatação personalizada dos dados do inquilino.

## 🚀 Conceitos Aplicados

- **Vetor de Objetos**: Manipulação de arrays de tipos referência (`Inquilino[]`).
- **Encapsulamento**: Uso de classe externa com atributos privados e métodos de acesso.
- **Sobrescrita de Método (Override)**: Implementação do método `toString()` para representação textual de objetos.
- **Lógica de Índices**: Mapeamento direto entre a entrada do usuário e a posição no array.
