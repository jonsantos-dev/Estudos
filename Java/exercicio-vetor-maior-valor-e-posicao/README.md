# Exercício: Maior Posição (Vetores)

Este programa lê N números reais e os armazena em um vetor. Em seguida, o sistema identifica qual é o maior elemento presente no vetor e exibe tanto o valor quanto o índice (posição) da primeira ocorrência desse valor.

## 📋 Funcionalidades

- Leitura e armazenamento de valores reais (`double`).
- Busca algorítmica do maior elemento dentro de uma estrutura de dados.
- Identificação do índice correspondente ao maior valor encontrado.
- Exibição de resultados via console.

## 🚀 Conceitos Aplicados

- **Vetores (Arrays)**: Armazenamento e manipulação de tipos primitivos de ponto flutuante.
- **Lógica de Comparação**: Uso de estruturas condicionais (`if/else`) para atualização dinâmica do maior valor.
- **Gerenciamento de Índices**: Rastreamento da variável de posição conforme o critério de maior valor é satisfeito.
- **Entrada de Dados**: Uso de `Scanner` com `Locale.US` para suporte a números decimais com ponto.
