# Exercício: Soma de Vetores (Desafio de Loop Único)

Este programa lê dois vetores de números inteiros (A e B) com o mesmo tamanho N. O sistema calcula a soma dos elementos correspondentes de cada vetor e armazena o resultado em um terceiro vetor (C), exibindo-o ao final.

## 📋 Funcionalidades

- Leitura de dois conjuntos de dados (Vetor A e Vetor B).
- Processamento otimizado: Leitura e cálculo realizados em uma estrutura de repetição personalizada.
- Geração do Vetor C, onde cada elemento `C[i]` é a soma de `A[i] + B[i]`.
- Exibição dos resultados somados.

## 🚀 Conceitos Aplicados

- **Arrays (Vetores)**: Manipulação simultânea de três vetores distintos.
- **Lógica de Índices**: Uso de deslocamento (`i - quantidade`) para gerenciar múltiplos arrays dentro de um único laço de repetição.
- **Acumuladores**: Atribuição e soma direta no vetor de destino para otimizar a memória.
- **Estruturas de Controle**: Uso estratégico de `if/else` para alternar entre os contextos de entrada dos vetores.
